# 2. Analyse du dossier `sprites/trainers`

## Base URL

```txt
https://play.pokemonshowdown.com/sprites/trainers/
```

## Résumé

Le dossier `sprites/trainers` contient un très grand nombre de sprites de dresseurs, personnages, champions, sbires, teams ennemies, PNJ et variantes issues de plusieurs générations ou sources.

Pour Pokegang, ce dossier est intéressant pour :

- créer des **agents recrutables** ;
- représenter des **sbires de gang** ;
- représenter des **dresseurs ennemis** ;
- créer des **boss de zone** ;
- créer des **rôles narratifs** : scientifique, biker, voleur, ninja, policier, marin, randonneur, etc. ;
- prototyper rapidement des portraits ou avatars sans créer tout de suite des sprites originaux.

⚠️ Pour un prototype perso, c'est très utile. Pour une version publique/distribuée, il faudra vérifier les droits, crédits et permissions.

### Agents

- Les sprites Rocket peuvent servir aux agents affiliés au gang du joueur.

- Les sprites `scientist`, `collector`, `ninjaboy`, `psychic`, `hexmaniac` peuvent représenter des agents spécialisés.

## 2.6 Attention droits / intégration

Pour le prototype :

- utiliser les URLs directes peut accélérer les tests ;
- mieux vaut prévoir une abstraction dans le code, par exemple `trainerSpriteId`, plutôt que coder les URLs partout ;
- garder une table `SPRITE_LIBRARY` centralisée.

Pour une version publique :

- vérifier les crédits ;
- éviter de modifier les sprites sans autorisation ;
- envisager de créer des sprites originaux inspirés du rôle mais pas copiés directement ;
- utiliser ces sprites comme placeholders temporaires.

# 4. Conclusion

Pour Pokegang, le meilleur usage serait :

1. utiliser les **FX Showdown** pour les effets rapides : zones, attaques, drops, shine, pokéball, coeur ;
2. utiliser les **sprites Rocket / Mafia** pour prototyper rapidement les agents et grades ;
3. utiliser les **sprites trainers** comme placeholders pour factions, rivaux, boss et événements ;
4. prévoir dès maintenant une structure propre pour remplacer plus tard par des assets originaux Pokegang.
